Project Detail

RetailFlow SQL Analytics

An end-to-end SQL analytics project on US retail data — covering star schema design, 30+ KPI queries, reusable views, and stored procedures to support business decision-making.

Open Repository Back to Portfolio

What It Does

Brief Overview

I modeled US retail data into a star schema, then wrote 30+ SQL queries covering sales KPIs, product performance, customer behaviour, and regional trends. Key outputs were packaged into views and stored procedures to make them reusable and easier for stakeholders to run independently.

Tools Used

Stack

  • SQL
  • Star Schema
  • Views
  • Stored Procedures

Why It Matters

Business Value

The project demonstrates how well-structured SQL work can replace ad-hoc reporting with repeatable, reliable KPI pipelines — the kind of foundation that data teams and business units can build on without starting from scratch each time.
