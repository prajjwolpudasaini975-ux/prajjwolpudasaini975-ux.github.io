Project Detail

Customer Churn Prediction and Segmentation

This project combines predictive modeling and business intelligence to identify likely churn behavior and segment customers into meaningful groups for better decision-making. I want to get more good at this for the time being.

Open Repository Back to Portfolio

What It Does

Brief Overview

I built an end-to-end analytics workflow using R and Power BI. The project includes a decision tree classifier for prediction, K-means clustering for segmentation, and a dashboard layer for interactive exploration.

Tools Used

Stack

  • R
  • Power BI
  • Decision Tree
  • K-means

Why It Matters

Business Value

The project shows that I can move from data preparation to modeling and then communicate results in a form that business stakeholders can actually use.
